# مرصد المخاطر المؤسسية

أداة عربية تفاعلية لتسجيل المخاطر المؤسسية وتقييمها ومعالجتها ومتابعتها وإعداد تقرير موجه لمجلس الإدارة.

تسترشد الأداة بمبادئ إدارة المخاطر في ISO 31000 وإطار COSO ERM، وتعمل بالكامل داخل المتصفح مع حفظ البيانات محليًا على جهاز المستخدم.
